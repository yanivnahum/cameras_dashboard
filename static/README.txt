Place a file named 'placeholder.jpg' in this directory. 
This will be used when a camera snapshot cannot be retrieved.

You can create a simple placeholder image with:
1. Any image editing software
2. Download a placeholder from a site like https://placekitten.com/320/240
3. Or use a command like: 
   wget -O placeholder.jpg https://placekitten.com/320/240

The recommended size is 320x240 pixels.
